## 工作流程

```
1. 市场调研 → 潜在 KOL 名单
2. 建档筛选 → 符合品牌调性
3. 外联接触 → 发送合作提案
4. 洽谈确认 → 报价和内容细节
5. 内容审核 → 确保符合品牌
6. 发布跟踪 → 监控内容和数据
7. 效果评估 → ROI 分析
```
